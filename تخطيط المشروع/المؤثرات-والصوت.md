# المؤثرات والصوت

## الملف

`game/js/sounds.js` — مولّد صوتي بـ Web Audio API (بدون ملفات MP3).

---

## المؤثرات المتوفرة

| الدالة | الوصف | الاستخدام |
|--------|-------|-----------|
| `SFX.click()` | نقرة زر | القائمة، الأزرار |
| `SFX.startRain()` | مطر مستمر | المقدمة — بدء القصة |
| `SFX.stopRain()` | إيقاف المطر تدريجياً | العودة للقائمة |
| `SFX.thunder()` | رعد | المقدمة — عشوائي كل ~9ث |
| `SFX.bossAppear()` | صوت مخيف منخفض | مشهد ظهور ملك الرعب |
| `SFX.bossAttack()` | هجوم ثقيل | مشهد إصابة أشرف |
| `SFX.playerHurt()` | ألم اللاعب | مشهد العائلة المقتولة |
| `SFX.arrow()` | إطلاق سهم | الغابة (مخطط) |
| `SFX.sword()` | ضربة سيف | الغابة (مخطط) |
| `SFX.hit()` | إصابة هدف | الغابة (مخطط) |
| `SFX.kill()` | موت عدو | الغابة (مخطط) |
| `SFX.xp()` | جمع نقاط | الغابة (مخطط) |
| `SFX.victory()` | انتصار (سلم موسيقي) | نهاية اللعبة (مخطط) |
| `SFX.defeat()` | هزيمة (نغمات هابطة) | موت البطل (مخطط) |
| `SFX.firework()` | ألعاب نارية | احتفال (مخطط) |

---

## ربط المؤثرات بالمقدمة

| حدث القصة | المؤثر |
|-----------|--------|
| ضغط «ابدأ القصة» | `click` + `startRain` + `thunder` بعد 1.5ث |
| مشهد #6 (ملك الرعب) | `bossAppear` |
| مشهد #4 (العائلة) | `playerHurt` |
| مشهد #7 (الهجوم) | `bossAttack` |
| رعد عشوائي أثناء القصة | `thunder` كل ~9ث (احتمال 35%) |
| العودة للقائمة | `click` + `stopRain` |

## المؤثرات البصرية — المقدمة

| التأثير | التفاصيل |
|---------|----------|
| **المطر** | 100 قطرة CSS متحركة |
| **البرق** | وميض أبيض/بنفسجي كل ~7ث |
| **الجسيمات** | 15 نقطة حمراء عائمة |
| **الكتابة** | حرف بحرف (45ms) + مؤشر وامض |
| **الانتقال** | fade overlay 800ms بين الصفحات |
| **الصور** | تبديل مع opacity 1.5ث + vignette |

## مؤثرات مخططة للمراحل القادمة

| المرحلة | المؤثرات المقترحة |
|---------|-------------------|
| الغابة | أجواء حشرات، رياح معركة، أصوات حيوانات |
| المدينة | أصوات سوق، حوار NPC |
| وادي الموت | عواصف، صدى |
| مملكة الظلام | موسيقى توتر، صرخات حراس |
| ملك الرعب | موسيقى زعيم، مؤثرات هجوم السيف والسوط |

## ملاحظات تقنية

- يُنشأ `AudioContext` عند أول تشغيل
- يُستأنف تلقائياً إذا كان `suspended` (قيود المتصفح)
- الأصوات مُولَّدة بـ oscillators + noise buffers — لا حاجة لملفات خارجية

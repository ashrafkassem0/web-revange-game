# تخطيط مشروع الانتقام

وثائق التخطيط الرسمية للعبة — تجمع القصة، المراحل، أنظمة اللعب، والشخصيات في مكان واحد.

## مسار اللعب الكامل

```
المقدمة (2D) → الغابة (2D) → المدينة (2D) → وادي الموت (2D) → مملكة الظلام (2D) → ملك الرعب
```

## ما هو مبني الآن

| المكوّن | المسار |
|---------|--------|
| القائمة الرئيسية | `game/index.html` |
| المقدمة (القصة) | `game/start/index.html` |
| حفظ التقدم | `game/js/shared.js` |
| المؤثرات الصوتية | `game/js/sounds.js` |

## التقنيات

- **HTML · CSS · JavaScript** فقط — لعبة **ثنائية الأبعاد** بالكامل
- **بدون** TypeScript أو Vite أو Three.js أو نماذج GLB
- الحفظ عبر `localStorage`

## ملاحظة

مجلد `senarios/` أصبح أرشيفاً — المصدر الرسمي للأفكار هو هذا المجلد (`تخطيط المشروع/`).

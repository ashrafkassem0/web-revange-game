# تخطيط مشروع الانتقام

وثائق التخطيط الرسمية للعبة — تجمع القصة، المراحل، أنظمة اللعب، والشخصيات في مكان واحد.

## مسار اللعب الكامل

```
المقدمة (2D) → الغابة (2D) ⇆ المدينة (2D) → وادي الموت (2D) → مملكة الظلام (2D) → ملك الرعب
```

> المدينة متاحة **اختيارياً** من الغابة في أي وقت (ذهاباً وإياباً).

## التقنيات

- **HTML · CSS · JavaScript** فقط — لعبة **ثنائية الأبعاد** بالكامل
- **Canvas 2D** للرسم والرسوم المتحركة
- **بدون** TypeScript أو Vite أو Three.js
- الحفظ عبر `localStorage`
- الخطوط: Google Fonts — Cairo

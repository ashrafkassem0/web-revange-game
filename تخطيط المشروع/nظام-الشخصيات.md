# نظام الشخصيات — الانتقام

## الملفات

| ملف | الوصف |
|-----|--------|
| `game/js/characters.js` | كل الشخصيات + stats + drops + مسارات GLB |
| `game/js/character-rules.js` | HP، القتل، الاكتساب 50%، الغنائm |
| `game/js/crafting.js` | فأس، سنارة، أسلحة قرn، أكل اللحم |

## القواعد

1. **HP = 0** → الموت
2. **القتل:** `attack >= defense` وإلا لا يُقتل
3. **الاكتساب:** 50% مهارات الضحية + HP
4. **السرعة:** +1 م/ث لكل كm جري
5. **التعديل:** غيّر الأرقام في `characters.js` فقط

## مواد التجميع

| مورد | المصدر |
|------|--------|
| لحم | أرنب، غزال |
| قرون | أrnب 50%، غزال 100% |
| أسنان | ذئb، ثعلb، أفعى، تمساح، خنزير |

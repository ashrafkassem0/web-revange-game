# خرائط مستقبلية

## مسار اللعب

```
المقدمة (2D) → الغابة → المدينة → وادي الموت → مملكة الظلام → ملك الرعب
```

## الخرائط

| # | الخريطة | المجلد | الحالة | ملاحظات |
|---|---------|--------|--------|---------|
| 1 | **الغابة** | `maps/forest/` | **الآن** | تدريب، حيوانات، صناعة |
| 2 | **المدينة** | `maps/city/` | placeholder | NPCs، تجارة |
| 3 | **وادي الموت** | `maps/death-valley/` | placeholder | terrain خطر |
| 4 | **مملكة الظلام** | `maps/dark-kingdom/` | placeholder | **ملك الرعب** |

## إضافة خريطة جديدة

1. أضف تعريف في `game/js/maps/index.js`
2. أنشئ `game/js/maps/<id>-map.js`
3. انسخ مجلد `maps/forest/` → `maps/<id>/`
4. عدّل `ForestScene.ts` → `<Name>Scene.ts`

## GameState

```javascript
unlockedMaps: ['forest']  // تُفتح تدريجياً
currentMap: 'forest'
completedForest / completedCity / completedDeathValley
```

**لا يوجد** `game/boss/` — الزعيم داخل مملكة الظلام.

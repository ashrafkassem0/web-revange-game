# خرائط مستقبلية — مرجع سريع (2D)

## مسار اللعب

```
المقدمة (2D) → الغابة (2D) → المدينة (2D) → وادي الموت (2D) → مملكة الظلام (2D) → ملك الرعب
```

## جدول الخرائط

| # | الخريطة | المجلد | الحالة | يفتح بعد | ملاحظات |
|---|---------|--------|--------|----------|---------|
| 1 | **المقدمة** | `game/start/` | **مبنية** ✅ | — | قصة 2D |
| 2 | **الغابة** | `game/forest/` | مخطط | `completedIntro` | تدريب، حيوانات، صناعة |
| 3 | **المدينة** | `game/city/` | مخطط | `completedForest` | NPCs، تجارة |
| 4 | **وادي الموت** | `game/death-valley/` | مخطط | `completedCity` | terrain خطر |
| 5 | **مملكة الظلام** | `game/dark-kingdom/` | مخطط | `completedDeathValley` | **ملك الرعب** |

## إضافة خريطة جديدة (عند التطوير)

1. أنشئ `game/<id>/index.html` بـ HTML · CSS · JavaScript
2. أضف صور/ sprites في `game/<id>/images/`
3. أضف تعريف الخريطة في `game/js/maps.js` (مخطط)
4. حدّث `GameState.getProgress()` و `navigateTo()` في `game/js/shared.js`
5. وثّق المرحلة في هذا المجلد

---

## GameState — المخطط الكامل

```javascript
// الحالة الحالية (مبنية)
completedIntro: false

// المخطط (للمراحل القادمة)
completedForest: false
completedCity: false
completedDeathValley: false
unlockedMaps: ['forest']
currentMap: 'forest'
heroStats: { stats, skills }
inventory: { stick, stone, meat, horn, teeth, tools }
totalDistanceRun: 0
currentPathId: 'forest-main'
pathCheckpoints: {}
completedChallenges: []
pathProgress: {}
```

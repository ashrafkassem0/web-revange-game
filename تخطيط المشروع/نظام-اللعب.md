# نظام اللعب

## القواعد الأساسية

### 1. الصحة (HP)
- عندما **HP = 0** → الموت
- أكل اللحم يستعيد HP:
  - لحم أرنب: +10
  - لحم غزال: +15

### 2. القتل
- يتطلب: **هجوم المهاجم ≥ دفاع الهدف**
- وإلا لا يُقتل العدو

### 3. الاكتساب (Absorb)
- عند قتل عدو: اكتساب **50%** من مهارات الضحية
- زيادة HP حسب نسبة الاكتساب (`absorbRatio: 0.5`)

### 4. السرعة
- كل **كيلومتر جري** يزيد السرعة بمقدار 1 م/ث (تخطيط طويل المدى)
- الجري يستهلك طاقة محدودة (~50 متر قبل التعب)

## الصناعة (Crafting)

| الأداة | المواد | الفائدة |
|--------|--------|---------|
| **فأس** | 1 حجر + 2 عصا | قطع الخشب |
| **سنارة** | 2 عصا + 2 سن | الصيد |
| **رمح قرن** | 1 قرن + 2 عصا | +15 هجوم |
| **سيف قرن** | 2 قرن + 1 حجر | +2 مهارة سيف |

### التحكم المقترح
- `F` — فتح قائمة الصناعة
- `Q` — أكل اللحم من المخزون

## المخزون (Inventory)

```javascript
{
  stick: 0,    // عصي
  stone: 0,    // حجر
  meat: 0,     // لحم
  horn: 0,     // قرون
  teeth: 0,    // أسنان
  tools: {}    // أدوات مصنوعة
}
```

## حفظ التقدم (GameState)

### المبني حالياً

```javascript
// game/js/shared.js
completedIntro: false   // true بعد انتهاء المقدمة
```

| الدالة | الوظيفة |
|--------|---------|
| `GameState.save(key, value)` | حفظ في localStorage |
| `GameState.load(key, default)` | قراءة |
| `GameState.getProgress()` | قراءة التقدم |
| `GameState.reset()` | مسح التقدم (لعبة جديدة) |
| `navigateTo(page)` | انتقال مع تأثير fade |

### المخطط للمراحل القادمة

```javascript
// التقدم العام
xp: 0
level: 1
completedForest: false
completedCity: false
completedDeathValley: false
unlockedMaps: ['forest']
currentMap: 'forest'

// البطل
heroStats: { stats: { hp, maxHp, speed, attack, defense }, skills: {} }

// المخزون
inventory: { stick, stone, meat, horn, teeth, tools }

// المسارات (الغابة)
totalDistanceRun: 0
currentPathId: 'forest-main'
pathCheckpoints: { 'forest-main': z }
completedChallenges: ['challenge-id']
pathProgress: { 'forest-main': z }
```

مفاتيح localStorage تبدأ بـ `revenge_`.

## الانتقالات بين المراحل

| من | إلى | الشرط |
|----|-----|-------|
| القائمة | المقدمة | لعبة جديدة أو أول تشغيل |
| المقدمة | القائمة | انتهاء القصة |
| القائمة | الغابة | `completedIntro` (مخطط) |
| الغابة | المدينة | `completedForest` (مخطط) |
| المدينة | وادي الموت | `completedCity` (مخطط) |
| وادي الموت | مملكة الظلام | `completedDeathValley` (مخطط) |
| مملكة الظلام | نهاية اللعبة | هزيمة ملك الرعب (مخطط) |

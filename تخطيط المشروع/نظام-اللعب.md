# نظام اللعب

## القواعد الأساسية

### 1. الصحة (HP)
- عندما **HP = 0** → موت البطل (شاشة موت مع خيار إعادة المحاولة أو القائمة الرئيسية)
- **استعادة HP:**
  - أكل لحم 🥩 (Q): +12 HP
  - أكل سمكة 🐟 (Q): +8 HP — إذا لم يوجد لحم
  - التقاط كرة حياة ❤️: يستعيد 12% من HP أقصى العدو
  - المعالج في المدينة: استعادة كاملة مقابل 2 لحمة، أو +20 مجاناً (راحة)

### 2. القتال والهجوم
- **السيف ⚔️:** هجوم قريب، يُفعَّل بـ E / Space / نقر أيسر
- **القوس 🏹:** يُطلق سهماً نحو الهدف المثبّت أو موضع الفأرة
- التبديل: مفاتيح 1/2 أو **عجلة الفأرة**
- **تثبيت الهدف:** نقر أيمن → رادار دوّار على العدو، السهم يتجه تلقائياً

### 3. الاكتساب (Absorb)
- عند قتل عدو: اكتساب **50%** من مهارات الضحية
- الاكتساب يزيد هجوم البطل تدريجياً

### 4. الطاقة والجري
- الجري (Shift) يستهلك شريط الطاقة
- الطاقة تتجدد تلقائياً عند التوقف

### 5. الاستفزاز
- أي حيوان فارّ يُصابُ بسهم وينجو → **يستشيط غضباً** ويهاجم
- يكتسب أوراً حمراء ويتسارع (+30% سرعة)

## التحكم الكامل

| المفتاح / الإدخال | الوظيفة |
|-------------------|---------|
| WASD / ↑↓←→ | تحرك |
| Shift | جري |
| E | هجوم / التقط / اقطع شجرة / ادخل بوابة مدينة |
| Q | أكل لحم أو سمكة |
| F | فتح قائمة الصناعة |
| R | بدء / سحب الصيد |
| 1 / 2 | تحديد السيف / القوس |
| عجلة الفأرة | تبديل سريع السيف ↔ القوس |
| نقر أيمن | تثبيت هدف على أقرب عدو |
| نقر أيسر | هجوم / إطلاق سهم نحو الهدف |
| Escape | إغلاق قائمة الصناعة |

## الصناعة (Crafting)

يُفتح بـ F، يُعرض بطاقات مع المتطلبات والحالة.

| الأداة | المواد | الفائدة |
|--------|--------|---------|
| 🪓 فأس | 1 حجر + 2 عصا | قطع الأشجار — يُمكّن مهارة قطع الخشب |
| 🎣 سنارة | 2 عصا + 2 أسنان | الصيد في البحيرات — يُمكّن مهارة الصيد |
| 🗡️ رمح القرن | 1 قرن + 2 عصا | +18 هجوم دائم |
| ⚔️ سيف القرن | 2 قرن + 1 حجر | +3 مهارة سيف |
| 🛡️ درع جلدي | 3 جلد + 1 عصا | +15 دفاع دائم |
| 🏹 سهام (10) | 2 عصا | قابلة للتراكم، تُضاف للمخزون |

## حفظ التقدم (GameState)

```javascript
// game/js/shared.js
GameState.save(key, value)       // حفظ في localStorage
GameState.load(key, default)     // قراءة
GameState.getHeroStats()         // { hp, maxHp, attack, defense, skills }
GameState.getInventory()         // المخزون
GameState.getCraftedItems()      // { axe, fishingRod, hornSpear, hornSword, leatherArmor }
GameState.getCurrentStage()      // المرحلة الحالية
GameState.saveForestState(data)  // حفظ حالة الغابة (موضع، قتلى، مسافة...)
GameState.loadForestState()      // استعادة حالة الغابة
GameState.reset()                // مسح كل شيء
```

### ما يُحفظ في الغابة
```javascript
{
  x, y,              // موضع البطل في العالم
  hp, stamina,       // الصحة والطاقة
  killCount,         // عدد القتلى
  distanceTraveled,  // المسافة بالبكسل
  xp,                // نقاط الخبرة
  collectedResources, // فهارس الموارد المُجمّعة
  choppedTrees,       // فهارس الأشجار المقطوعة
  seenIntro: true,    // لتجاوز المقدمة عند الاستئناف
  savedAt             // توقيت الحفظ
}
```

## الانتقالات بين المراحل

| من | إلى | الشرط |
|----|-----|-------|
| القائمة | المقدمة | لعبة جديدة |
| المقدمة | القائمة | انتهاء القصة |
| القائمة | الغابة | `completedIntro = true` |
| الغابة | المدينة | **اختياري** — اقترب من بوابة الجنوب (E) |
| المدينة | الغابة | اقترب من بوابة الشمال (E) |
| الغابة | المدينة (دائمة) | `completedForest = true` (مخطط) |
| المدينة | وادي الموت | `completedCity` (مخطط) |
| وادي الموت | مملكة الظلام | `completedDeathValley` (مخطط) |
| مملكة الظلام | نهاية اللعبة | هزيمة ملك الرعب (مخطط) |

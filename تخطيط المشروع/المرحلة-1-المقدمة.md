## الإعداد القصصي

- **المكان:** سفوح الجبل — منطقة جبلية ممطرة بجوار غابة فيها حيوانات مفترسة وأرانب برية
- **البطل:** أشرف (11 سنة)
- **العائلة:** الأم، الأب، الأخت آية
- **العدو:** ملك الرعب — رداء أسود، عينان حمراوان، سيف أسود ملطخ بالدماء

## المؤثرات البصرية

- خلفية عنوان مع تعتيم (`bg_title_screen.png`)
- مطر متحرك (100 قطرة)
- برق متقطع
- جسيمات حمراء عائمة
- تأثير كتابة حرف بحرف مع مؤشر وامض
- انتقال صور المشاهد مع vignette
- زر **تخطي** أعلى اليسار

## المؤثرات الصوتية (مرتبطة بالمشهد)

| المشهد | الصوت |
|--------|-------|
| بدء القصة | مطر + رعد |
| ظهور ملك الرعب | `bossAppear` |
| مشهد العائلة المقتولة | `playerHurt` |
| هجوم ملك الرعب | `bossAttack` |
| الأزرار | `click` |
| العودة للقائمة | إيقاف المطر |

انظر [المؤثرات-والصوت.md](./المؤثرات-والصوت.md) للتفاصيل.

---

## واجهة المستخدم

1. **شاشة العنوان** — عنوان اللعبة + زر «ابدأ القصة»
2. **شاشة القصة** — صندوق نص + صورة خلفية للمشهد
3. **زر الإجراء** — «العودة للقائمة الرئيسية» (يظهر بعد انتهاء القصة)

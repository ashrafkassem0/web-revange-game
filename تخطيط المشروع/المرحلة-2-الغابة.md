# المرحلة 2 — الغابة (2D)

## الحالة: مخطط (غير مبنية)

| البند | القيمة |
|-------|--------|
| **المسار المقترح** | `game/forest/index.html` |
| **النوع** | خريطة تفاعلية 2D — HTML · CSS · JavaScript + Canvas |
| **الإكمال** | `completedForest = true` |
| **الانتقال** | → خريطة المدينة (المرحلة 3) |
| **يفتح بعد** | إكمال المقدمة (`completedIntro`) |

---

## السياق القصصي

بعد دفن عائلته وقسمه بالانتقام، دخل أشرف الغابة وحده.
لم يعد طفلاً يجمع الحطب فقط — بل محارباً مبتدئاً يتدرب قبل مواجهة ملك الرعب.
الغابة واسعة، مليئة بالموارد والمخاطر. كل خطوة تقرّبه من الانتقام.

---

## أهداف المرحلة

1. التدرب على القتال ضد الحيوانات
2. جمع الموارد (عصي، حجر، لحم، قرون، أسنان)
3. صناعة أدوات (فأس، سنارة، أسلحة قرن)
4. تقوية المهارات والإحصائيات بالاكتساب من القتلى
5. الاستعداد للرحلة القادمة: المدينة → وادي الموت → مملكة الظلام

---

## العالم — خريطة الغابة

### المكان

- **المساحة:** خريطة 2D من أعلى (top-down) أو side-scrolling
- **نقطة البداية:** قرب الحافة الجنوبية للغابة (حيث دخل أشرف)
- **المناخ:** ضباب خفيف، إضاءة نهارية

### التضاريس والعناصر

- أرض خضراء مع حشائش وصخور (صور PNG أو tiles)
- أشجار كثيفة قرب نقطة البداية
- بحيرات (شمال، جنوب، بركة وسطية) — فيها سمك وتماسيح على الشاطئ
- موارد متفرقة: عصي وحجر للصناعة
- طبقات parallax للخلفية (أشجار بعيدة، سماء)
- كهف فرعي (مسار `forest-cave`)

### مناطق الحيوانات

| المنطقة | الحيوانات |
|---------|-----------|
| المساحات المفتوحة | أرانب برية، غزلان |
| الغابة الكثيفة | ذئاب، ثعالب، خنازير برية |
| التلال الصخرية | أفاعي، دببة |
| أعالي الأشجار | نسور |
| شواطئ البحيرات | غوريلا، تماسيح |
| داخل الماء | سمك |

---

## أشرف في الغابة

- **العمر:** 11 سنة
- **الأسلحة:** قوس وسهم، سيف الأب
- **الحركة:** مشي | جري (طاقة محدودة ~50 متر)
- **المهارات الابتدائية:** رماية، سيف، سباحة
- **القاعدة:** كل كيلومتر جري يزيد السرعة (تخطيط طويل المدى)

---

## التحكم (مخطط — 2D)

| المفتاح | الفعل |
|---------|--------|
| `W A S D` أو الأسهم | التحرك |
| `Shift` | الجري |
| `Space` | القفز / التفاعل |
| `F` | قائمة الصناعة |
| `Q` | أكل اللحم |
| `E` أو زر الماوس | هجوم / تفاعل |
| `1` / `2` | التبديل بين القوس والسيف |

---

## واجهة اللاعب (HUD)

- الحياة والسرعة والمسافة المقطوعة
- شريط طاقة الجري
- خريطة مصغّرة (موارد، بحيرات، موقع اللاعب)
- عداد التحديات لإكمال الغابة

---

## المشاهد القصصية المقترحة

1. **دخول الغابة** — أشرف يتأمل قبور عائلته من بعيد
2. **أول قتال** — أرنب بري (ذكرى الصيد القديم)
3. **أول صناعة** — صنع فأس أو سنارة
4. **لقاء خطر** — ذئب أو دب يختبر مهاراته
5. **نهاية المرحلة** — أشرف يغادر الغابة نحو المدينة

---

## شروط إكمال المرحلة (مقترحة)

- قتل عدد معيّن من الوحوش (مثلاً 10)
- قطع مسافة معيّنة (مثلاً 3 كم)
- إكمال تحديات محددة (مثلاً 2)
- صناعة فأس وسنارة

```javascript
GameState.save('completedForest', true);
```

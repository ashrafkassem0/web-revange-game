# خريطة الغابة

## الحالة: قيد التنفيذ 🌲

- **المسار:** `game/maps/forest/`
- **المحرk:** Three.js + TypeScript + Vite
- **المساحة:** 5 كm² (2500×2000 م)
- **البيانات:** `game/js/maps/forest-map.js`

## آليات

| مفتاح | فعل |
|-------|-----|
| WASD | تحريك |
| Shift | جري → أنيمiشن Run |
| Space | قفz → Jump |
| F | صناعة |
| Q | أكل لحم |

## الحيوانات (12)

أrnب، ذئb، غزال، ثعلb، أفعى، دب، نsr، غوريلا، خنزير، تمساح، سمك

## الإكمال

`completedForest = true` — 3 كm + 10 حيوانات *(قابل للتعديل في GAME_CONFIG)*

## التشغيل

```bash
npm install
npm run dev
```

افتح بعد المقدمة أو مباشرة: `http://localhost:5173`

## النماذج

ضع `hero.glb` في `game/assets/models/` — انظر `threejs-animations.md`

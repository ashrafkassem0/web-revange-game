# الانتقام (Revenge)

لعبة مغامرات وانتقام في المتصفح — قصة عربية تفاعلية **ثنائية الأبعاد**.

> *قصة أشرف — طفل يتدرب في الغابة لينتقم من ملك الرعب.*

**التقنية:** HTML5 Canvas 2D · Vanilla JS · CSS3 · صفحات متعددة (ليست SPA)  
**الحفظ:** `GameState` / `SaveManager` عبر `localStorage`  
**الصوت:** مؤثرات إجرائية في `sounds.js` (مستوى صوت / كتم)

---

## التشغيل السريع

افتح في المتصفح:

```
game/index.html
```

أو خادم محلي للواجهة:

```bash
python -m http.server 8080 --directory game
# ثم: http://localhost:8080/
```

خادم الـ API (اختياري — مصادقة / حفظ سحابي / لوحة متصدرين):

```bash
cd server
npm install
npm run migrate
npm run dev
```

## ما يعمل الآن

### القائمة والمقدمة
- **القائمة** (`game/index.html`) — لعبة جديدة / متابعة، مع عناصر تحكم بالصوت
- **المقدمة** (`game/start/`) — قصة تفاعلية ثم الانتقال للغابة

### الغابة (`game/forest/`)
- خريطة كبيرة مع تضاريس مسبقة التصيير، مياه متحركة، دورة نهار/ليل
- قتال (سيف / قوس)، صيد، قطع أشجار، صناعة، حقيبة، بناء (مواقد / أكواخ / أسوار)
- بوابة جنوبية للمدينة مع حفظ عند الانتقال
- **دورة النهار/الليل (مهمة 005):**
  - أطوار `فجر / نهار / غسق / ليل` مع طبقة ظلام وثقوب إضاءة (موقد / كوخ / مشعل اللاعب)
  - الساعة تتوقف مع الإيقاف والقوائم الحاجبة (صناعة، حقيبة، نوم، بناء)
  - الوقت يُحفظ ويُستعاد مع لقطة الغابة (`clockMinutes` / `dayCount`)
  - نجوم خفيفة ليلاً، وتخفيف الظلام قرب الكوخ
- **الطقس (مهمة 006):**
  - حالات: صحو / مطر خفيف / مطر غزير / ضباب / عاصفة (`forest-weather.js`)
  - قطرات Canvas (~50–150) + صوت مطر/رعد عبر `SFX` + وميض واهتزاز في العاصفة
  - أيقونة بجانب الساعة؛ المؤقت يتوقف مع القوائم؛ يُحفظ في لقطة الغابة
- **الموقد (مهمة 007):**
  - قائمة `E` عند الموقد: حفظ / راحة / طبخ / إضرام (عودان)
  - يحفظ `meta.lastCampfire`؛ الأعداء القريبون يمنعون الراحة/الحفظ
  - المطر الغزير/العاصفة قد تُطفئ المواقد؛ الوحوش تبتعد قليلاً عن النار
- **موارد متقدمة (مهمة 008):**
  - جمع `عشبة` و`عسل` من نقاط على الخريطة (`E`)
  - وصفات: مرهم أعشاب / منشط منعش (مستهلكات شفاء)
  - المفاتيح متزامنة عبر `GameState` بين الغابة والمدينة
- **البناء (مهمة 009):**
  - وضع البناء `B`: سياج / بوابة / موقد / كوخ مع معاينة وتكلفة
  - `E` يفتح/يغلق البوابة أو يصلح المباني المتضررة (الأقرب أولاً)
  - المباني تُحفظ مع اللقطة (`hp` / `lit` / `open` / `uid`)
- **ذكاء الأعداء (مهمة 010):**
  - فكّ التعليق عند الانحشار + نبض أحمر عند الاستفزاز
  - تنبيه `!` واستدعاء خفيف للذئاب القريبة؛ ضبط مدى المطاردة
- **إكمال الغابة (مهمة 011):**
  - شروط: 10 قتلى + 3000 مسافة + فأس وسنارة
  - بانر + بوابة جنوبية؛ مغادرة مبكرة لا تُسجّل الإكمال
  - مشهد تخرج قصير ثم المدينة مع الإبقاء على لقطة الغابة
- **واجهة HUD:**
  - **يسار:** الموارد (صف واحد) ثم حالة البطل
  - **منتصف:** التوقيت + أيقونة الطقس
  - **يمين:** خريطة مصغّرة + حفظ/صوت بجانبها، ثم قائمة التحديات تحتها

### المدينة (`game/city/`)
- تاجر / معالج / حداد
- مخزون مشترك مع الغابة عبر `GameState` (يشمل اللحم/السمك النيء والمطهو)
- العودة للغابة من البوابة الشمالية

### الأسس المكتملة (مهام 001–011)
| المهمة | المحتوى |
|--------|---------|
| 001 | `SaveManager` + `GameState` بإصدارات وفتحات حفظ |
| 002 | موجّه مشاهد خفيف (`SCENES` / `navigateToScene`) بدون SPA |
| 003 | مستوى صوت رئيسي/مؤثرات/أجواء + كتم (`M`) |
| 004 | مزامنة المخزون والمصنوعات بين الغابة والمدينة |
| 005 | صقل دورة النهار/الليل (إيقاف الساعة، حفظ الوقت، مشعل/نجوم/كوخ) |
| 006 | نظام طقس الغابة (مطر/ضباب/عاصفة + SFX + HUD) |
| 007 | قائمة الموقد (حفظ / راحة / طبخ / إضرام) |
| 008 | موارد متقدمة (عشبة / عسل + مرهم / منشط) |
| 009 | صقل البناء (بوابة E، إصلاح، حفظ المباني، تغذية راجعة) |
| 010 | صقل ذكاء الأعداء (انحشار، استفزاز، تنبيه، قطيع خفيف) |
| 011 | تدفق إكمال الغابة → المدينة (CFG + تخرج + لقطة) |

الخادم: مصادقة + API حالة اللعبة (مهام 034–036) جاهز؛ ربط العميل (037–038) لاحقاً.

## أزرار التحكم (الغابة)

| المفتاح / الإدخال | الوظيفة |
|-------------------|---------|
| WASD / ↑↓←→ | تحرك |
| Shift | جري |
| E | تفاعل / التقط / اقطع / بوابة / موقد / كوخ / إصلاح / هجوم قريب |
| Q | أكل |
| F | قائمة الصناعة |
| I | الحقيبة |
| B | وضع البناء |
| R | صيد (إلقاء / سحب) |
| P | حفظ يدوي |
| M | كتم / إلغاء كتم الصوت |
| 1 / 2 | سيف / قوس |
| عجلة الفأرة | تبديل سيف ↔ قوس |
| نقر أيمن | تثبيت هدف |
| نقر أيسر | هجوم / سهم |
| Escape | إغلاق القوائم |

## التقنيات

| الطبقة | التقنية |
|--------|---------|
| العالم | HTML5 Canvas 2D + `requestAnimationFrame` |
| الواجهة | HTML/CSS (قوائم، HUD، نوافذ) |
| المنطق | Vanilla JavaScript |
| الحفظ المحلي | `localStorage` عبر `GameState` |
| الصوت | Web Audio API (إجرائي) |
| الخادم | Node.js · Express · SQLite · JWT |
| الخط | Cairo (Google Fonts) |
| البناء | بدون حزمة واجهة — ملفات HTML مباشرة |

**لا يُستخدم Pixi.js** — الرسم Canvas 2D فقط.

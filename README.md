# الانتقام (Revenge)

لعبة مغامرات وانتقام في المتصفح — قصة عربية تفاعلية **ثنائية الأبعاد**.

> *قصة أشرف — طفل يتدرب في الغابة لينتقم من ملك الرعب.*

---

## المميزات الحالية

### مرحلة الغابة (`game/forest/index.html`)
- **خريطة 3200×3200** مع تضاريس إجرائية (عشب، غابة كثيفة، بحيرات، صخور، رمال)
- **تكبير ×2** — كل الكائنات والخريطة بحجم واضح
- **تضاريس سلسة** بدون تربيع — عبر لوحة مسبقة التصيير مع بقع ألوان عضوية
- **موجات مياه متحركة** مقصوصة بمنطقة البحيرة بدون تربيع مرئي
- **حركة البطل** WASD / الأسهم مع جري (Shift)، رسوم متحركة للأرجل والذراعين
- **عجلة الفأرة** للتبديل بين السيف والقوس
- **النقر الأيمن** لتثبيت الهدف على عدو قريب مع رادار دوّار وشريط HP
- **النقر الأيسر** لإطلاق السهم نحو الهدف المثبّت تلقائياً
- **السيف** (مفتاح E أو Space أو النقر الأيسر) للهجوم القريب
- **القوس والسهام** بمدى محدود — سهام قابلة للصنع من العصي

### الأعداء
- غزال، أرنب، ذئب، خنزير بري، دب، غوريلا (ضفدع مسموم)
- الحيوانات **تهرب** من البطل، وتنقلب وتهاجمه عند الإصابة بسهم
- الأعداء **لا يدخلون الماء** — يتجنبون البحيرات
- عند الموت: يسقطون **غنائم على الأرض** (لحم، قرون، أسنان، جلد، كرة حياة)
- كرة الحياة تعيد HP للبطل عند التقاطها

### نظام التقاط الأشياء
- كل الأشياء على الأرض تُلتقط بمفتاح **E** عند الاقتراب
- تُعرض مطالبة `[E] خذ` مع توهج نابض فوق كل عنصر
- العناصر تبقى على الأرض إذا تجاهلها البطل

### الصيد
- اصنع **سنارة صيد** (🎣) من عصيتين وأسنانين
- اقترب من الماء واضغط **R** لإلقاء الخيط
- انتظر إشارة العضّ ثم اضغط **R** مجدداً للسحب
- السمك قابل للأكل (Q) لاستعادة HP

### قطع الأشجار
- اصنع **فأس** (🪓) من حجر وعصيتين
- اقترب من شجرة واضغط **E** — يحتاج عدة ضربات
- تسقط الأشجار عصياً (🪵) كغنائم على الأرض
- الأشجار كبيرة مع جذوع مفصلة وجذور وأوراق متعددة الطبقات

### الصناعة (مفتاح F)
| العنصر | المواد |
|--------|--------|
| 🪓 فأس | ×1 حجر + ×2 عصا |
| 🎣 سنارة | ×2 عصا + ×2 أسنان |
| 🗡️ رمح القرن | ×1 قرن + ×2 عصا |
| ⚔️ سيف القرن | ×2 قرن + ×1 حجر |
| 🛡️ درع جلدي | ×3 جلد + ×1 عصا |
| 🏹 سهام (10) | ×2 عصا |

### الحفظ التلقائي
- يحفظ كل 30 ثانية وعند إغلاق المتصفح
- يستعيد الموضع، HP، المهارات، المخزون، الأشجار المقطوعة، الموارد المُجمّعة
- يستأنف اللعبة مباشرة بدون مقدمة عند العودة

### المدينة (`game/city/index.html`)
- **بوابة في جنوب الخريطة** (مسار رملي + قوس حجري متوهج)
- التنقل: اقترب من البوابة واضغط **E** للدخول — والعودة من البوابة الشمالية داخل المدينة
- **التاجر** 🧑‍💼 — تبادل الجلد / اللحم / السمك / الأسنان بسهام
- **المعالج** 🧑‍⚕️ — شفاء كامل مقابل لحم، شفاء جزئي مقابل سمك، راحة مجانية +20 HP
- **الحداد** ⚒️ — عرض المعدات المصنوعة

## أزرار التحكم

| المفتاح / الإدخال | الوظيفة |
|-------------------|---------|
| WASD / ↑↓←→ | تحرك |
| Shift | جري |
| E | هجوم / التقط / اقطع شجرة / ادخل بوابة |
| Q | أكل لحم أو سمكة |
| F | فتح قائمة الصناعة |
| R | بدء / سحب الصيد |
| 1 / 2 | تحديد السيف / القوس |
| عجلة الفأرة | تبديل سريع السيف ↔ القوس |
| نقر أيمن | تثبيت هدف (يتحول للقوس تلقائياً) |
| نقر أيسر | هجوم / إطلاق سهم نحو الهدف |

## التشغيل

افتح مباشرة في المتصفح:

```
game/index.html
```

أو مرحلة الغابة مباشرة:

```
game/forest/index.html
```

### خادم محلي (اختياري)

```bash
python -m http.server 8080 --directory game
# ثم: http://localhost:8080/forest/index.html
```

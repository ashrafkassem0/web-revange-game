# الانتقام (Revenge)

لعبة مغامرات وانتقام في المتصفح — قصة عربية تفاعلية تبدأ بمقدمة **ثنائية الأبعاد**، ثم تنتقل إلى **خرائط ثلاثية الأبعاد** مبنية بـ **Three.js** و **TypeScript**.

> *قصة أشرف — طفل يتدرب في الغابة لينتقم من ملك الرعب.*

---

## المميزات

- **مقدمة سينمائية** — قصة RTL بالعربية مع مؤثرات صوتية
- **خريطة الغابة 3D** — مساحة 5 كم² (أشجار، صخور، بحيرات)
- **شخصيات متحركة** — Idle / Walk / Run / Jump مع انتقالات ناعمة (crossfade)
- **دعم GLB** — استيراد شخصيات من Mixamo/Blender (Skinned Animation)
- **نظام شخصيات** — HP، مهارات، غنائm، صناعة (فأس، سنارة، أسلحة)
- **خرائط مستقبلية** — المدينة، وادي الموت، مملكة الظلام (ملك الرعب)

## التشغيل المحلي

### المتطلبات

- [Node.js](https://nodejs.org/) 20.19 أو أحدث (مطلوب لـ Vite 8)
- npm

### 1. استنساخ المشروع

```bash
git clone https://github.com/ashrafkassem0/web-revange-game.git
cd reveange
```

### 2. تثبيت الاعتماديات

```bash
npm install
```

### 3. تشغيل خريطة الغابة (وضع التطوير)

```bash
npm run dev
```

يفتح المتصفح على `http://localhost:5173`

### 4. المقدمة (بدون Vite)

افتح الملف مباشرة في المتصفح:

```
game/start/index.html
```

بعد انتهاء القصة اضغط **«ادخل خريطة الغابة»**.

## طريقة اللعب

### المقدمة

- اقرأ القصة أو اضغط **تخطي**
- اضغط **ادخل خريطة الغابة**

### خريطة الغابة

| المفتاح | الفعل |
|---------|--------|
| `W A S D` أو الأسهم | التحرك |
| `Shift` | الجري (Run) |
| `Space` | القفز (Jump) |
| `F` | قائمة الصناعة *(قريباً في HUD)* |
| `Q` | أكل اللحم *(قريباً)* |

## الشخصيات والخرائط

### خرائط اللعبة

| الخريطة | الحالة | الوصف |
|---------|--------|--------|
| 🌲 **الغابة** | ✅ متاحة | تدريب، حيوانات، صناعة |
| 🏙 **المدينة** | 🔜 قريباً | NPCs وتجارة |
| 💀 **وادي الموت** | 🔜 قريباً | terrain خطر |
| 🌑 **مملكة الظلام** | 🔜 قريباً | مواجهة **ملك الرعب** |

### إضافة شخصية Mixamo

1. حمّل شخصية من [Mixamo](https://www.mixamo.com) (Idle, Walk, Run, Jump)
2. صدّر إلى **GLB** عبر Blender
3. ضع الملف في `game/assets/models/hero.glb`
4. راجع [`تخطيط المشروع/threejs-animations.md`](تخطيط%20المشروع/threejs-animations.md)

> بدون ملف GLB، اللعبة تعرض مجسمًا بديلًا بحركة إجرائية.

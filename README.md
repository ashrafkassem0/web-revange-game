# الانتقام (Revenge)

لعبة مغامرات وانتقام في المتصفح — قصة عربية تفاعلية **ثنائية الأبعاد**.

> *قصة أشرف — طفل يتدرب في الغابة لينتقم من ملك الرعب.*

---

## المميزات

- **مقدمة سينمائية** — قصة RTL بالعربية مع مؤثرات صوتية
- **حفظ التقدم** — تذكّر إكمال المقدمة عبر localStorage
- **بدون بناء** — HTML و CSS و JavaScript فقط، لا حاجة لـ npm أو Vite

---

| الطبقة | التقنية |
|--------|---------|
| المقدمة | HTML · CSS · JavaScript |

## التشغيل

### فتح مباشر

افتح في المتصفح:

```
game/index.html
```

أو ابدأ من المقدمة مباشرة:

```
game/start/index.html
```

### خادم محلي (اختياري)

إذا احتجت خادماً محلياً (مثلاً لتجنب قيود `file://`):

```bash
# Python
python -m http.server 8080 --directory game

# أو Node (بدون تثبيت مشروع)
npx serve game
```

ثم افتح `http://localhost:8080/index.html`.

# المراحل 3–5 — المستقبل

## الحالة: مخطط فقط

---

## المرحلة 3 — المدينة

| البند | القيمة |
|-------|--------|
| **المجلد المقترح** | `game/city/` |
| **المساحة** | ~2 كم² |
| **يفتح بعد** | `completedForest` |
| **الإكمال** | `completedCity = true` |

### الأفكار

- NPCs للتجارة والحوار
- شراء معدات ومواد نادرة
- معلومات عن ملك الرعب ومملكة الظلام
- مهام جانبية تمهّد لوادي الموت
- لا أعداء قتاليون رئيسيون — تركيز على التفاعل الاجتماعي

### المشهد الافتتاحي (مقترح)

أشرف يصل إلى المدينة بعد تدريبه في الغابة، يبحث عن من يوجّهه نحو عدوّه.

---

## المرحلة 4 — وادي الموت

| البند | القيمة |
|-------|--------|
| **المجلد المقترح** | `game/death-valley/` |
| **المساحة** | ~8 كم² |
| **يفتح بعد** | `completedCity` |
| **الإكمال** | `completedDeathValley = true` |

### الأفكار

- تضاريس خطرة (منحدرات، وديان، براكين خامدة)
- أعداء أقوى من الغابة
- موارد نادرة لصناعة أسلحة متقدمة
- طقس قاسٍ (عواصف رملية، ضباب)
- اختبار نهائي لمهارات أشرف قبل المواجهة الأخيرة

### المشهد الافتتاحي (مقترح)

المدينة تُرسل أشرف عبر وادي الموت — الطريق الوحيد إلى مملكة الظلام.

---

## المرحلة 5 — مملكة الظلام + ملك الرعب

| البند | القيمة |
|-------|--------|
| **المجلد المقترح** | `game/dark-kingdom/` |
| **المساحة** | ~3 كم² |
| **يفتح بعد** | `completedDeathValley` |
| **الزعيم** | ملك الرعب (`terrorKing`) |

### الأفكار

- قلعة أو مملكة مظلمة بإضاءة حمراء
- حراس قبل الوصول للزعيم
- **ملك الرعب** — نفس الشخصية من المقدمة
  - رداء أسود، عينان حمراوان، سيف أسود
  - HP 1000، مهارات: سوط، سيف، قوة جسدية
- مواجهة نهائية متعددة المراحل
- نهاية القصة: الانتقام لعائلة أشرف

### المشهد الافتتاحي (مقترح)

أشرف يدخل البوابة السوداء لمملكة الظلام — يتذكر ليلة مقتل عائلته.

### نهاية اللعبة (مقترحة)

- هزيمة ملك الرعب → مشهد انتصار (`victory` صوتياً)
- أشرف يقف عند قبور عائلته — الحلقة مغلقة

---

## مسار فتح الخرائط

```javascript
unlockedMaps: ['forest']           // بعد المقدمة
// ثم تدريجياً: city → death-valley → dark-kingdom
```

**لا يوجد** مجلد `game/boss/` منفصل — الزعيم داخل خريطة مملكة الظلام.

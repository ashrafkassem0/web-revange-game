# نماذج الشخصيات (GLB)

ضع ملفات Mixamo/Blender هنا:

| الملف | الشخصية |
|-------|---------|
| `hero.glb` | أشرف — شخصية Erika Archer (Mixamo) + أنيمiشن |
| `rabbit.glb` | الأرنب البري |
| `wolf.glb` | الذئb |
| `deer.glb` | الغزال |
| `terror-king.glb` | ملك الرعب (مملكة الظلام) |

## Mixamo

1. اختر شخصية على [mixamo.com](https://www.mixamo.com)
2. حمّل الأنيمiشن: Idle, Walking, Running, Jump
3. صدّر FBX → Blender → Export glTF Binary (.glb)
4. سمِّ الـ clips كما في `characters.js` → `animations`

**بدون GLB:** الل game تعرض شخصية بديلة (مجسم بسيط) مع حركة إجرائية.
